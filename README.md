 Project Overview

This project implements the Algorithmic Efficiency & Recursion Toolkit (AERT) in Python.

The toolkit demonstrates:

Implementation of a Stack ADT

Recursive Factorial

Recursive Fibonacci (Naive + Memoized)

Tower of Hanoi (with trace for N = 3)

Recursive Binary Search

Time and Space Complexity Analysis

The goal of this assignment is to understand recursion, algorithmic paradigms, and complexity analysis using practical implementations.

 Key Observations

Naive Fibonacci grows exponentially (O(2ⁿ)).

Memoization reduces Fibonacci complexity to O(n).

Binary Search works in O(log n).

Tower of Hanoi requires (2ⁿ − 1) moves.

Stack operations run in O(1) time.
